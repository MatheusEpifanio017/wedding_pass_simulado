 - Descrição breve do projeto:
   Esse é o backend do projeto "Wedding Pass" que implementa a estrutura de uma API do sistema conectada com o banco de dados.

 - Tecnologias utilizadas:
   Vs code, node.ts, express, typeorm, reflect_metadata, dbeaver.

- Pré requisitos:
  typescript, mariaDB, node v22.14.0, mysql 8.0. 

- Passo a passo (como criar e popular tabelas):
  Abrir dbeaver e colocar o comando da pasta script.sql para popular as tabelas.

- Como subir o servidor localmente:
  Para subir abra o vscode e digite no terminal "npm start", depois no navegador, cole "http://localhost:3000/"
